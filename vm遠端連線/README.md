## VM遠端連線
*如果滑鼠卡在vm裡可以按ctrl回到實體機*

1.先將虛擬機關閉

去設定>顯示>遠端顯示>按啟用伺服器

*埠號可以自己設*

2.開啟虛擬機

3.去實體機的命令提示字元

圈起處代表尋找實體機已用埠號
